本番2028年1月 共通テスト → 2月 総合問題▶

1日の組み方の原則

  1. 英語は毎日（単語＋もう1つ）。1日も空けない。
  2. 1日＝英語2コマ＋他科目1〜2コマが基本。
  3. 平日2〜3時間、休日・長期休み5〜7時間。
  4. 週の最終日は「間違えた所だけ」復習日。

🏛 総合問題（独自100点・高3夏から本格化）▶

形式：120分で日本語長文・英語長文・英作文の3題。統計図表・データ分析・推論・論述。日英の情報を統合して考える力が核心。

主教材
公式サンプル＋過去問（河合塾の解答例も活用）｜P2で眺める→P3で年度順→P4で120分実戦
土台
英語ルート（長文＋英作文）／開発講座の記述／数ⅠA「データの分析」／社会の背景知識＋時事全部ここに合流する

⚠️ 記述は必ず第三者の添削（学校の先生・塾）。独学で点数化しづらい唯一の領域。

現在地とギャップ

  1. 英語 48 → 進研65〜70へ（最優先で立て直し）
  2. 国語 48 → 進研65〜70へ（読解力が全科目に波及）
  3. 数学 48 → 数ⅠAで共テ8割（基礎から）
  4. 社会 約60（推定） → 共テ9割（強み・要実測）
進研は河合・駿台より高めに出る。高3では必ず河合全統で「本当の現在地」を測る。判定より弱点発見ツールとして使う——E判定が続いても、潰す穴が見えていれば前進している。

検定の方針

漢検 → やらない。早慶一般で不使用・時間対効果が低い。
数検 → やらない。数ⅠAは共テ過去問で直接叩く方が速い。
英検 → 使う。到達度マーカー＋併願保険。高2で2級 → 高3前半で準1級挑戦。早稲田他学部の4技能利用の保険にもなる。

絶対ルール

  1. 1冊を完璧に、2〜3周。薄く広くは厳禁。最大の敵は「やった気」。
  2. 英語は毎日。単語・解釈・長文を途切れさせない。
  3. 順番厳守。早慶レベル教材は高3前半以降。土台なしで難問に行くと時間が溶ける。
  4. 終わった教材は復習用に残す。捨てない。
  5. 模試のたびに計画を更新。速ければ前倒し、遅れたら★を絞って基礎優先。
